## 🏛️ What is This Showcase?

The showcase website is a beautiful, high-performance **Glassmorphic Portfolio Dashboard** built with custom vanilla CSS and dynamic JavaScript state management. It provides a visual-rich overview of every containerization milestone achieved over my 4-week learning path.

## 🛠️ Key Features

### 1. 📅 4-Week Interactive Timeline
*   **Decoupled Navigation**: Switch between weeks seamlessly using an dynamic tab controller.
*   **Deep Git Explorer Linking**: Every weekly block includes a dedicated glassmorphic **"Explore Directory"** button that redirects you directly to the corresponding source code folder in the main branch.

### 🔌 2. Live System Telemetry Simulation
*   A responsive container network visualizer illustrating network traffic packets flowing through an isolated bridge subnet.
*   Dynamically maps simulated data flow from **Nginx Web Gateways** ➡️ **Express Backend Core APIs** ➡️ **Redis Memory Caches** ➡️ **PostgreSQL/MongoDB databases**.
*   Built with mobile-first media queries to stack beautifully into an active pipeline on vertical screens.

### 🎨 3. Premium Glassmorphic UI
*   Clean, harmonized dark-mode aesthetic utilizing rich CSS gradients, customized box shadows, HSL tailored border styling, and beautiful Google Font integrations.
*   Micro-animations on hover states for buttons, pipeline flows, and navigation menus.
